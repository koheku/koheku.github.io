Photo sharing game

Share photos with friends as if they were trading cards.

How does it work ?

The idea is to replicate the behaviour of trading cards.

Take a photo, add a caption and send it to friends. If they re-send it to someone, they lose it. Just like a trading card !

Technical details (v2.0)

I wrote this app in Objective-C for iOS7+.

It was built using the VIPER architecture for cleanliness, testability and reusability of the code.

The app consumes a JSON REST API and beta testing was done using HockeyApp.

  • AFNetworking
  • Mantle
  • Overcoat
  • AsyncDisplayKit
  • Pop
  • XCTest
  • VIPER
  • Autolayout
  • GCD
  • HockeyApp
  • REST
