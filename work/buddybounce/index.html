Buddybounce

(v1.0 on the Appstore)

Modern day fan club

The Buddybounce app enables you to share and discover content around celebs that you follow on the Buddybounce platform.

How does it work ?

The Buddybounce app uses the buddybounce.com platform to deliver content (photos, tweets, videos) relevant to the celebs you are interested in. It aggregates the entire fan experience (facebook, instagram, tumblr, twitter etc...) in a single place.

Technical details

Version 1.0 was written in Objective-C for iOS7+ using Xcode 6.

The app consumes a JSON REST API. It was developed using TDD to ensure quality and stability.

Beta testing was done using HockeyApp.

  • Autolayout
  • JSON

  • Google analytics
