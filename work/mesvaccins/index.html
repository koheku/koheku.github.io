MesVaccins

(v2.0 currently waiting for review)

A smart immunisation record

The MesVaccins app makes sure that you are properly protected against vaccine preventable diseases.

How does it work ?

The MesVaccins app interacts with the mesvaccins.net platform to deliver personalised vaccine recommendations tailored for you. It uses an expert system that computes these recommendations according to rules established by the French government.

It also enables you to anonymously store your vaccinations and share them with healthcare professionals.

Originally developed for iOS5+ in 2012, the app saw a complete overhaul for iOS7+ in early 2015 with version 2.0. The app is only in french for the time being.

Technical details (v2.0)

I wrote this app in Objective-C for iOS7+ using Xcode 6. It is a complete overhaul of the MesVaccins v1.0 iOS app that I built in 2012.

It was built using the VIPER architecture for cleanliness, testability and reusability of the code.

The app consumes a JSON REST API.

Beta testing was done using HockeyApp.

  • Core Data
  • Autolayout
  • VIPER
  • HockeyApp
  • REST API
  • JSON
  • Kiwi testing framework
  • GCD
